# 데이터 표현법
 > c언어에서 데이터를 표기하는 방법을 리터럴(literal)이라고 하며, 값을 그대로 입력한 것을 의미합니다. (10, 11, 12 등 순수한 값만을 리터럴이라고 한다.) <br>
 > 정수의 경우에는 그대로 기입합니다. ex) 10	 <br>
 > 실수의 경우에는 마지막에 f를 붙여줍니다. ex) 10.0f <br>
 > 문자의 경우에는 작은 따옴표 안에 기입합니다. ex) 'a' <br>
 > 문장에 경우에는 큰 따옴표 안에 기입합니다. ex) 'apple' <br>

# 변수
> c언어에서 특정 하나의 데이터를 사용자가 이름지어서 저장하는 방법입니다.(혹은 저장하는 공간)
> 변수에 저장되어있는 데이터의 값은 일반적으로 언제든지 변형이 가능합니다.
> c언어에서의 변수는 변수마다 정해진 데이터의 형태(자료형)이 존재하며, 각 변수는 할당된 값을 가지고 있습니다.

<hr/>

# 자료형
> 프로그램이 데이터를 판단하는 기준, 데이터의 형태를 의미함
> 대표적인 자료형 <br>

|이름|표현하는 형태|용량|표현 범위|
|-----|-----|-----|-----|
|int|정수|4byte|-2147483648 ~ 217483647|
|float|실수| 4byte|3.4 * 10 -38제곱 ~ 3.4 * 10 + 38제곱 --> 소수점 이하 6자리까지 정확하게 표현|
|char|문자|1byte|-128 ~ 127|

# count
> 변수에 대한 선언 <br>
> 변수를 선언할 경우 자료형의 크기만큼 메모리 공간을 확보하여 그 위치에 변수의 이름을 지정하게 됩니다.

+ 변수를 만들 때 지켜줄 이름 규칙 <br>
	1. 대소문자를 구분합니다 ex) a 와 A는 서로 다름 <br> 
	2. 변수의 이름은 숫자가 맨 앞에 나롱 수 없습니다. ex) int 1a; <--안됨 , int a1; <--됨 <br>
	3. 변수의 이름에 언더바 이외의 특수문자 사용이 불가능 합니다. (예외) _, $는 c언어에서 사용 가능 <br>
	4. 변수의 이름으로 키워드(keyword)를 사용할 수 없습니다.

```c 
#include <stdio.h>

int main()
{
	  int count; 
 //(자료형) (변수명);

	count = 5;
	//변수에 대한 초기화
	//변수명 = 타입에 맞는 값;
	
	printf("현제의 카운트는 %d입니다.\n", count);

	
	//변수는 선언과 초기화를 동시에 진행할 수 있습니다. 
	float pi = 3.14f;

	//데이터의 형태가 같다면, 여러 개를 한번에 선언할 수 있습니다. 
	int a, b, c;


	//선언과 초기화를 동시에 여러 개를 진행할 수 있습니다. 
	int number3 = 50, number4 = 40;

	//c언어에서 10진수 이외의 8진수나 16진수를 지정할 수 있습니다. 
	int octo = 015; //숫자 앞에 0이 붙으면 8진수
	int hexa = 0x6; //숫자 앞에 0x이 붙으면 16진수

	printf("8진수 octo = %o\n", octo);
	printf("16진수 hexa = %x\n", hexa);
	printf("8진수 표기법 출력 hexa = %#o\n", octo);
	printf("16진수 표기법 출력 hexa = %#x\n", hexa);
	printf("8진수 표기법 출력(8자리) octo : %0.8o\n", octo);
	printf("16진수 표기법 출력(8자리) hexa : %0.8x\n", hexa);

	return 0;
}
```

# 입력
> 입력(input) <br>
> 사용자가 직접 값을 입력하고 그 값을 특정 위치에 전달합니다.일반적으로는 변수를 만들고, 그 변수에 입력한 값을 적용하는 방식으로 사용합니다.

+ getchar() <br>
	> 키보드로부터 하나의 문자를 읽어내는 함수 <br>
	> 이 기능은 버퍼를 비우는 용도로도 사용됩니다. <br>
	> 버퍼(buffer) : c언어에서 입력을 진행할 경우 바로 값을 넘기는게 아니라 임시로 데이터를 저장하는 별도의 공간에 값을 넣고 전달하는 방식 <br>
	> 문자를 다 받기 때문에 enter같은 기능도 값으로 남게 됩니다. <br>

+ putchar(문자형 데이터) <br>
	> 전달받은 문자 하나를 화면에 출력하는 기능

+ gets(), puts() <br>
 > 주로 문자열에 대한 입출력을 진행할 때 편하게 사용가능 <br>
	  문자열이란 char 형태의 데이터를 묶음으로 표현한 것을 의미합니다. <br>
 > 문자열 만드는 방법 <br>
 > char 변수명[문자의 개수];

+ 주의사항 <br>
 > c언어에서 한글을 입력하는 경우 한글 하나당 2칸을 소모함 <br>
 >	현재 word는 10개의 단어를 사용가능하고 한글은 5개의 단어까지 입력 가능합니다. <br>
	  단, 입력을 진행할 경우 문자의 개수보다 1개정도 적게 작성합니다. ex) 10칸 사용할 경우 한글은 1칸 빼고 2로 나눠서 4칸밖에 사용못함 <br>
	  따라서 영단어는 9글자까지, 한글은 4글자까지 가능 

+ 사용예시

```c

#define _CRT_SECURE_NO_WARNINGS
//프로그램에서의 안전검사를 하지 않도록 설정하는 코드
//이 코드를 추가할 경우 visual studio 내에서 scanf를 사용할 수 있습니다.

#include <stdio.h>

int main()
{
	int number;
	printf("input the number : ");
	scanf_s("%d", &number); 
	//visual studio에서 사용하는 경우라면 scanf_s로 입력하여 오류를 막아야함
	printf("number : %d\n", number);

 getchar(); // 키를 하나 받겠습니다. (이걸로 남아있는 enter를 처리합니다.)
	
	char key;
	printf("키를 하나 입력해주세요");
	key = getchar();
	putchar(key);

 getchar(); // 위에서 입력한 key 다음 동작인 enter키에 대한 값을 받아줍니다.

 char word[10];
	printf("단어를 입력해 주세요 >> ");
	gets(word); //문자열 입력 gets는 c언어에서만 사용 가능
	puts(word); //전달한 문자열 출력
	
 char item[10];
	printf("아이템의 이름을 입력해주세요 >> ");
	scanf("%s", item);
	//문자열을 받을 경우에는 item 앞에 &를 붙이지 않아도 됩니다.
	//%s는 문자열을 받는 지정자 입니다.
	printf("아이템의 이름 : %s\n", item);


	return 0;
}
```
